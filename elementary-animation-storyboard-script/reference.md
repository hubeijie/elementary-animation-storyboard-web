- 脚本 **画面描述** 里的 **`*复用场景-*` / `*复用道具-*`** 应被 `animation-handoff-assets` 汇总进美术表对应行，避免改名。

## 【n】与复用标记（摘自范本习惯）

- **【n】**：同一镜或同一教学块内，台词（或逐字稿）与画面（或动效）用相同编号逐步推进。
- **\*复用\***：便于美术统筹与资产复用追踪；保持与脚本原文一致，不要随意缩写或改写文件名式命名。

## 黄金样例（建议粘贴脱敏真稿）

Agent 生成脚本/教学知识表时，会优先模仿这里的**语气、【n】密度、复用标记写法**。按下面做即可：

1. 在 Cursor 中打开本文件：`~/.cursor/skills/elementary-animation-storyboard-script/reference.md`（或通过命令面板 **Go to File** 搜 `elementary-animation-storyboard-script`）。
2. 滚动到本节下方的 **代码块**（一对 \`\`\`markdown … \`\`\`）。
3. **删掉**块内以 `<!--` 开头的那行注释。
4. 从你平时的 **脚本表** 或 **教学知识表** 里，选 **5～15 行最满意、最典型** 的片段：
   - 校名、学生真名、未公开链接、内部项目代号等先改掉或打码；
   - 若从 Excel 复制，在表里对齐列后，可粘贴到 [表格生成器](https://www.tablesgenerator.com/markdown_tables) 转成 Markdown 表，或手工写成与上文「空模板」相同的列顺序。
5. 把转好的 Markdown **粘进该代码块**，保存文件（`Cmd+S`）。
